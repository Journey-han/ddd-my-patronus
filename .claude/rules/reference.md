## 2. Dynamic Grid (dynamic-grid-rose.vercel.app)

> ⚠️ JavaScript 렌더링 기반 SPA로, 정적 콘텐츠 추출이 제한적. v0(Vercel) 기반 프로젝트 분석 및 실제 접속 경험을 바탕으로 작성.

### 📐 레이아웃

| 항목 | 분석 |
|------|------|
| **구조** | 대시보드 형태의 **동적 그리드 레이아웃**. 고정된 구조 없이 사용자가 직접 배치를 조정할 수 있는 인터랙티브 그리드 |
| **그리드 시스템** | CSS Grid 기반으로, 각 셀이 다수의 그리드 컬럼/로우를 span할 수 있음. 반응형으로 모바일 1컬럼, 태블릿 2컬럼, 데스크탑 3~4컬럼으로 자동 조정 |
| **카드 배치** | 각 차트/플롯 카드가 독립적인 그리드 아이템으로 작동. 크기 조절 시 주변 카드가 자동으로 재배치(reflow) |
| **사이드바** | 접이식(collapsible) 사이드바에 플롯 옵션을 배치. 메인 그리드 영역과 분리된 제어 패널 구조 |
| **동적 확장** | 카드를 빈 영역으로 드래그하면 새로운 행이 자동 생성되고, 나머지 카드 크기가 자동 조정 |

### ✨ 화면 효과

| 효과 | 설명 |
|------|------|
| **드래그 앤 드롭** | @dnd-kit 라이브러리 기반. 카드 헤더를 잡고 드래그하면 그림자 + 스케일 변화와 함께 이동. 드롭 시 framer-motion으로 부드러운 레이아웃 전환 애니메이션 |
| **리사이즈** | 카드 모서리와 엣지에 리사이즈 핸들 배치. 드래그로 크기 조절 시 하이라이트 링이 표시되고, 주변 요소가 실시간으로 재배치 |
| **호버 피드백** | 카드에 마우스를 올리면 그림자가 깊어지며, 드래그 가능 영역에는 "move" 커서와 "(Drag to move)" 힌트 텍스트 표시 |
| **레이아웃 애니메이션** | framer-motion의 layout animation으로 카드 위치·크기 변경 시 부드러운 트랜지션. 갑작스러운 점프 없이 유기적으로 재배열 |
| **토스트 알림** | 레이아웃 변경(카드 이동, 자동 리사이즈) 시 하단에 토스트 알림으로 사용자에게 변경 사항 전달 |
| **차트 인터랙션** | Chart.js 기반의 Bar, Line, Pie, Scatter 차트가 각 카드 내부에서 자체적인 호버/클릭 인터랙션 제공 |

### 🎨 디자인 시스템

| 요소 | 상세 |
|------|------|
| **컬러** | 라이트 테마 기반의 중립적 UI. 화이트 배경 + 소프트 그레이 보더. 차트별로 구분되는 데이터 컬러 팔레트 |
| **컴포넌트 라이브러리** | shadcn/ui + Radix UI 기반. Card, Dialog, Dropdown Menu, Context Menu, Collapsible, Tabs, ScrollArea 등 일관된 컴포넌트 세트 |
| **타이포그래피** | 시스템 폰트 기반의 깔끔한 산세리프. 데이터 가독성을 우선하는 실용적 타이포 체계 |
| **인터랙션 패턴** | 우클릭 컨텍스트 메뉴(리사이즈/삭제 옵션), 드래그 핸들, 리사이즈 핸들 등 데스크탑 네이티브 앱에 가까운 조작 방식 |
| **상태 표현** | 드래그 중(반투명 + 그림자), 리사이즈 중(하이라이트 링), 호버(그림자 증가) 등 명확한 상태별 시각 피드백 |
| **톤** | 순수 기능적·도구적. 장식을 최소화하고 데이터 시각화와 레이아웃 조작에 집중 |

## 핵심 인사이트

**StringTune**은 라이브러리 자체가 곧 데모인 사이트로, 스크롤 애니메이션과 텍스트 스플리팅의 가능성을 극대화하여 보여줍니다. 기술 도구임에도 강한 내러티브와 감성적 비주얼을 가진 점이 차별화 포인트입니다.

**Dynamic Grid**는 기능 중심의 도구형 UI로, 사용자의 자유로운 레이아웃 조작이 핵심입니다. shadcn/ui 기반의 일관된 컴포넌트 시스템과 framer-motion의 레이아웃 애니메이션이 조작감에 정교함을 더합니다.

**Jesko Jets**는 럭셔리 브랜드의 정석적 웹 표현으로, 다중 레이어 패럴랙스와 절제된 애니메이션으로 품격을 전달합니다. 기술적으로는 가장 보수적이지만, 브랜드 경험 측면에서 가장 완성도 높은 연출을 보여줍니다.
